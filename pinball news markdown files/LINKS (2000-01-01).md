**Pinball News is pleased to recommend these other fine sites from around the world. They are listed alphabetically.**

Machine and parts suppliers have now moved to the much larger list found in our new [The Directory](https://www.pinballnews.com/site/2000/01/01/the-directory/) section.

If you have a website you think we should include, just send us a message using the CONTACT US box on the right side of this page.

**PINBALL MANUFACTURERS**

|  |

|  |  |
| --- | --- |
| **[Dutch Pinball](http://www.dutchpinball.com/)** | Netherlands |
| [Dutch Pinball](http://www.dutchpinball.com/) | Dutch Pinball made a big impression when they announced their first title and took it to a number of pinball shows. The Big Lebowski went into production in Summer 2015, promising to the the first of a series of machines from the Utrecht-based company. Following legal issues, production has now resumed with the remaining pre-ordered machines being manufactured alongside new sales. |

|  |  |
| --- | --- |
| **[Jersey Jack Pinball](http://www.jerseyjackpinball.com/)** | USA |
|  | Jersey Jack Pinball’s site serves as an introduction to the new company and the members of the team building their games.  On the website you can also read the update messages sent to subscribers to their mailing list and find out how to buy their pinball machines, parts, add-ons and request service assistance. |
|  |

|  |  |
| --- | --- |
| **[Spooky Pinball](http://www.spookypinball.com/)** | USA |

Their first game was America’s Most Haunted of which 250 units were built. This was followed by Rob Zombie’s Spookshow International and many subsequent titles, with more pinball projects in the pipeline. |
|  |

|  |  |
| --- | --- |
| **[Stern Pinball](http://www.sternpinball.com/)** | USA |
|  | Stern’s site provides details of the company’s current and archived products, providing links to pictures, videos, brief rules and software downloads.  It also includes details on how to buy their products, where you can play them, videos of their most recent games and a number of blogs with links to press coverage of pinball events. |

**ASSOCIATIONS**

|  |  |
| --- | --- |
| **[Flipper SV Austria](http://www.flipper-sv.at/)** | Austria |
|  | The Flipper Sportverein Austria website is dedicated to promoting pinball ownership, playing in leagues and competitions across Austria, as well as providing news and information about events in the pinball world. There is also a discussion forum, a pinball locator, pictures of game rooms, and a newsletter. |
|  |

|  |  |
| --- | --- |
| **[International Flipper  Pinball Association – I.F.P.A.](http://www.ifpapinball.com/)** | USA |
|  | Apart from being the home of the WPPR ranking system with around 30,000 registered players, and the website of the IFPA World Championships, the IFPA also promotes competitive pinball around the world and organises a number of tournaments through their Country Directors spread across the world. |
|  |

|  |  |
| --- | --- |
| **[Nederlandse Flipper Vereniging](https://www.nfvpinball.nl/)** | Netherlands |
| [De Nederlandse Flipper Vereniging](https://www.nfvpinball.nl/) | The NFV, or Dutch Pinball Association, publishes the quarterly Spinner magazine for its members, holds the annual Dutch Pinball Open, runs a club house in Veenendaal which is also used for the Dutch Pinball Masters, organises regional leagues across the country, and much more. |
|  |

**INFORMATION**

|  |  |
| --- | --- |
| **[Aussie Pinball](http://www.aussiepinball.com/)** | Australia |
|  | Formed in 2008, Aussie Pinball has quickly become the home of the Australian Pinball community. Collectors and hobbyists from Australia and around the world share stories, document restorations and technical tips on all things pinball. The preservation of pinball and sharing the knowledge is their focus. |
|  |

|  |  |
| --- | --- |
| [**Flipper France**](http://www.flipperfrance.com/) | France |
|  | A great French language site with lots of articles about restoration, electronics and other practical information. To get the most out of the site you need to register but that gives you access to the forum area where many of the other sections reside. |
|  |

|  |  |
| --- | --- |
| **[Flipper Pinball Fan](http://www.flipper-pinball-fan.be)** | Belgium |
|  | This Belgian site (don’t worry – there’s an English version too) contains a good range of hints, tips and ideas from fixing circuit boards to making your own custom game parts. With clear pictures and waveforms, it’s a great site for pinball game repairs and modifications. |
|  |

|  |  |
| --- | --- |
| **[Flippers](http://www.flippers.be)** | Belgium |
|  | Another great Belgian site with an English language version. Flippers.be features a large number of detailed articles about all aspects of pinball from show reports, book and game reviews, through a full restoration guide, to the pinball basics like how to take out the backglass and remove the head. |
|  |

|  |  |
| --- | --- |
| **[Internet Pinball Database](http://www.ipdb.org/)** | USA |
|  | A great resource detailing just about every pinball game ever made. There is a comprehensive search engine and it includes links to external sites with rulesheets and other information about the games. |
|  |

|  |  |
| --- | --- |
| **[MoPinball](http://www.mopinball.com/)** | USA |
|  | Missouri Pinball provides some  good solid information for new pinball owners. Greg Peck’s site also  features a growing number of game pages for detailed descriptions, pictures  and links. |
|  |

|  |  |
| --- | --- |
| [**Penny Machines**](http://www.pennymachines.co.uk/) | UK |
|  | Available in Flash and non-Flash versions, this UK site is dedicated to the beautifully crafted and ornate coin operated amusements from the start of the last century up until 1970. There’s a comprehensive library section, a discussion forum and you can even try your hand at some of the games in the arcade section. |
|  |

|  |  |
| --- | --- |
| **[Pinball Eric](http://pinballeric.com/)** | USA |
|  | The home of the Pinball Price Guide is also filled with information and links about pinball’s history from its earliest days with the transformation from Bagatelle, to the present day and the latest Stern Pinball machines.   Plus, of course, there are links to purchase the latest edition of Pinball Eric’s price guide. |
|  |

|  |  |
| --- | --- |
| **[Pinball Gallery](http://www.pinball-gallery.com/)** | France |
|  | Pinball Gallery is the website of Raphael Lankar’s pinball museum just north of Paris and contains a many pictures of the pinballs, bingos, slots and other coin-op products in his collection. This includes many rare and unusual models as well as assorted foreign conversions of US games not often seen on display. |
|  |

|  |  |
| --- | --- |
| [**Pinball Map**](http://www.pinballmap.com/) | USA |
|  | Pinball Map is *the* place to find pinball machines to play on location all around the world. Updated by users, the website and companion app is mostly US-focussed but the 7,500+ locations listed also cover Canada, the UK, Australia, Finland and Japan. Add and update the machines near you to keep Pinball Map accurate and help support pinball on location. |
|  |

|  |  |
| --- | --- |
| [**Pinball Owners**](http://www.pinballowners.com/) | Italy |
|  | If you own a pinball machine, this is the place to register it and get in contact with fellow owners. You can leave interesting details about each game, add photos, rank its condition and let potential buyers know if you’re thinking of selling. Registered users can also display their location, games owned and preferred type of machines. The site is available in English and Italian versions. |
|  |

|  |  |
| --- | --- |
| **[Pinball Price.com](http://www.pinballprice.com/)** | USA |
|  | Pinballprice.com is the place to find out how much pinball machines have sold for in the US at auctions and on Ebay. Historic data goes back to 2009 so you can see how prices have varied over the years. Plus you can submit your own sales results and find out about upcoming auctions. |
|  |

|  |  |
| --- | --- |
| **[Pinball Prices.com](http://www.pinballprices.com/)** | USA |
|  | Regularly updated with the latest sales data, Pinballprices.com is where you can find out the prices of pinball machines sold for in the US on Ebay, on Pinside and at auctions. There is also a links section for parts and other pinball information sites. |
|  |

|  |  |
| --- | --- |
| **[Pin Repair](http://www.pinrepair.com/)** | USA |
| [Pinball Repair](http://www.pinrepair.com/) | When it comes to maintaining and repairing coin-operated games of all kind, Clay Harrell’s Pin Repair site contains a veritable wealth of information. There are specific guides for common problem areas such as flippers which go into quite some depth and are best printed out for reference. |
|  |

|  |  |
| --- | --- |
| **[Pinball Wizard](http://hem.bredband.net/b257182/Pinball.htm)** | Sweden |
|  | A great resource for real pinball fans. This site contains loads of sound samples and tunes, pictures, and perhaps most importantly, details of all the hidden features found in so many games. The site is also home to the owners lists for The Addams Family Gold and Earthshaker! |
|  |

|  |  |
| --- | --- |
| **[Pinballz](http://www.pinballz.net/)** | Germany |
|  | Pinballz.net calls itself an international community and it is the place for pinball fans from many countries to meet and talk pinball. Messages are mainly in German but English posts are also encouraged. With around 40 different discussion areas there’s bound to be something to interest you. |
|  |

|  |  |
| --- | --- |
| **[Pinformer](http://www.pinformer.co.uk/)** | UK |
|  | Pinformer maintains a list of pinball machines on site in the UK. You can search the database for either a particular game or location. If you spot or play an unlisted pinball game you can rank it, describe the condition and let people know how much it costs for a game. Pinformer also allows private collections to be added so you can find fellow pinball collectors around the country. |
|  |

|  |  |
| --- | --- |
| **[Pinside](http://www.pinside.com/)** | Netherlands |
|  | The relaunched Pinside features a mix of general articles about pinball, details of specific games and a bunch of customisable features.  But the most popular section is the forums which has become the favoured place for pinball fans to discuss the game, new machines, new products and technical repairs. |
|  |

|  |  |
| --- | --- |
| **[Tilt!](http://www.tilt.it/flipper_pinball/)** | Italy |
|  | A bi-lingual site, Tilt! is the home of pinball in Italy. The site contains masses of information about Italian games, game conversions and has ROMs for Zaccaria games. There are some behind-the-scenes pictures, classified ads and polls together with personal collection information. |
|  |
